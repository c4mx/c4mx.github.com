My blog is born ！

Posted on 2019-02-08 | Edited on 2019-02-10

I’m so happy that I finally have a place to hold all my posts.

This blog is powered by a blog framework named Hexo. However, what I first tried is Jekyll, a very popular blog framework and has more support on Github Pages. Unfortunately, several things keep it away from me:

  • The commands of the jekyll involve many different concepts which confuse me: bundle, gem, and jekyll. I was trying to understand each concept and their relationship. For example, to launch the server, you’d better use bundle exec jekyll server, instead of jekyll server.
  • One theme can be installed on system level (with sudo), or it can be installed in local ~/.gem folder, otherwise it can also be installed in the project folder. Often it takes me so much time to find out where one of my gem package is.
  • I feel I need to read a lot of documentation to learn how to customize a theme.

When I tried Hexo, everything becomes easy for me:

  • Use only one command hexo, to create a new post, to run the server, to publish on Github Pages, etc.
  • A theme is installed directly inside your blog project.
  • I begin to customize the theme by reading the theme config file. I don’t even need to read the doc.

Yes, I like what said by Leonardo da Vinci:

Simplicity is the ultimate sophistication.
